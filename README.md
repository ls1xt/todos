# todos-general

Shared scripts, hooks, and templates for a voice-driven todo system. See [PLAN.md](PLAN.md) for the full design.

```
Hotkey → Murmur (STT) → Claude Code → todos/add → .md file → git commit
```

Included as a git submodule (`todos/`) in the [business](../business) and [personal](../personal) todo repos.
